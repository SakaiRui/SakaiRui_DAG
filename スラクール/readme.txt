開発の工夫
・ゲームについて
　ゲームを考える時にもただやりたいことをやるのではなく、
より面白くするためにはどうするかを考えてゲームを制作しました。
具体的には2Dのただ左右に飛び跳ねるゲームにするのではなく、
カメラの向きを変えれるようにすることで、
奥や手前にも飛び跳ねることが出来るようにした所や、
それによりただ壁に引っ付いて、反対方向に飛ぶことが出来るだけでなく、
プレイヤーを基準とした壁以外の三方向に跳ねることが出来るようにしました。

・判定の取り方
　気になったという理由でまだ授業で習ってなかったレイを独学で勉強して
壁に引っ付く判定で使用しました

・マウスカーソルについて
　マウスカーソルの見た目を変更したり、
追従してパーティクルエフェクトを生成させたり、
座標を取得してその座標によってプレイヤーの跳ねる高さや勢いが変わるようにするなど、
マウスカーソルに関する様々な処理を独学でやりました。

・スライム感
　スライムっぽさをより出すために、見た目だけ跳ねる時に潰れたり、
着地のタイミングで音が出たり、壁にペタッとくっつくようにしました。
